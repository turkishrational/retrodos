Retro DOS v5.0 (Modified PCDOS 7.1) Disk Images

(FD = floppy disk, HD = hard disk, HD32 = 32MB, RD5 = Retro DOS 5, R5 = Retro DOS 5 floppy)
.IMG = disk(ette) image file. (Floppy disk images are for 3.5" 1.44 MB diskettes.)
(R5MCONF = CONFIG.SYS and AUTOEXEC.BAT are set for multiple startup configuration menu)
Note: Hard disk images are needed for (proper) configured/menu startup. (I mean startup with the sample configuration.)
((see "qemu_link.txt" file for QEMU demo of Retro DOS v5.))

(Retro DOS v4 hard disk images can be used in addition to the sample Retro DOS v5 hard disk image.)
